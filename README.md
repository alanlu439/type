# Type

Type is a modern, simple typewriter simulator. It lets visitors type normally while their words appear as uneven ink on a vintage paper surface with an animated full keyboard, moving typebars, and browser-native typewriter sound effects.

## Run Locally

This is a static website. Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Features

- Live paper rendering from a real text input
- Visible keyboard that moves with physical key presses and on-screen clicks
- PDF import for text-based documents
- PDF export of the current page
- Ink and paper tone controls
- Reset controls
- Sound on/off with key, return, delete, and bell effects
- One-page responsive typewriter workspace
