# Type

Type is a modern, simple typewriter simulator. It lets visitors type normally while their words appear as uneven ink on a vintage paper surface, with browser-native typewriter sound effects.

## Run Locally

This is a static website. Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Features

- Live paper rendering from a real text input
- Ink and paper tone controls
- Reset and plain-text export
- Sound on/off with key, return, delete, and bell effects
- Responsive landing and typewriter experience
